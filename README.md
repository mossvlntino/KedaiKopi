# ☕ KedaiKopi Website

## 📖 Overview
A responsive frontend web interface designed for a specialty coffee shop. This project focuses on delivering a clean, intuitive user experience (UX) for browsing menus and locating the storefront. It served as a practical exercise in simplifying visual layouts and ensuring consistent design across different screen sizes.

## 🛠️ Focus Areas
* **UI/UX Design:** Creating an accessible, highly readable menu structure.
* **Frontend Development:** HTML, CSS, and basic JavaScript interactions.
* **QA Testing:** Manual responsive layout testing across mobile and desktop views.
